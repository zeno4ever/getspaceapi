Get the open/closed status of a hackerspace by the spaceapi.

== Description ==

Get the status of a hackerspace by reading the JSON api file.

== Installation ==

1. Upload `getspaceapi` directory to the `/wp-content/plugins/` directory

2. Activate the plugin through the 'Plugins' menu in WordPress

3. Configure the Get Spaceapi plugin under the dashboard/settings.

After activating the plugin you can enter the JSON location. If you don't know what 
this should be check spaceapi.org.

You have to fill in at least the open / closed tekst fields. You can enter just a plain tekst or HTML code.
